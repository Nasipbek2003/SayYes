# Templates

This directory holds the **data-driven template schemas** that power the
invitation scenario engine.

In the MVP, the three template schemas (`simple-date`, `story-fork`,
`event-rsvp`) live here as TypeScript files. The runtime engine interprets
these declarative schemas to render screens and validate guest responses.

Schema types and the concrete templates are implemented in tasks 3.1–3.2.
This file is a placeholder establishing the directory as part of the project
scaffold (task 1).
