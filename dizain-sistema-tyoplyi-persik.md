# Дизайн-система — «Тёплый персик»

> Базовая визуальная тема сервиса приглашений, выбранная как основная. Документ описывает цвета, типографику, иконки и компоненты — чтобы дизайнер и разработчик могли работать по единому стандарту.

---

## 1. Цветовая палитра

| Назначение | Цвет | HEX | Где использовать |
|---|---|---|---|
| Фон (основной) | Кремовый | `#FFF6EF` | Фон всех экранов приглашения |
| Фон вторичный / плашки | Светлый персик | `#FFD9C2` | Кружки-аватары, декоративные сердечки, неактивные карточки |
| Акцент / декор | Персиково-коралловый | `#FF9A76` | Иконки, второстепенные акценты, ховер-состояния |
| Кнопка «Да» / основное действие | Коралл | `#E8625A` | Главные CTA-кнопки, выбранная карточка, прогресс |
| Текст основной | Тёмно-коричневый | `#3D2C2A` | Заголовки, основной текст |
| Текст вторичный | Приглушённый коричневый | `#8A736C` | Подзаголовки, описания |
| Текст на кнопке | Светлый кремовый | `#FBEAE6` | Текст внутри кораллового CTA |
| Граница карточки (неактивной) | Светло-серо-бежевый | `var(--color-border-tertiary)` | Обводка карточек по умолчанию |
| Граница карточки (выбранной) | Коралл | `#E8625A` | Обводка 1.5–2px у выбранного варианта |
| Подпись бренда | Бледный беж | `#C9B3A8` | Мелкий текст «сделано в [бренд]» |

**Дополнительная палитра для негативного/отказного состояния (Экран 2 сценария):**
| Назначение | HEX |
|---|---|
| Фон иконки-настроения | `#FFE3D2` |
| Иконка/акцент «грусти» | `#D9897A` |

**Правило сочетаний:**
- Не больше 2-3 цветов на одном экране одновременно (фон + 1 акцент + 1 текстовый тон).
- Коралл `#E8625A` — единственный цвет для главного призыва к действию; не дублировать его в декоре, чтобы кнопка не терялась.
- Текст всегда тёмно-коричневый или приглушённо-коричневый — никогда чёрный (`#000`) и никогда серый (`#999`), это ломает тёплый тон палитры.

## 2. Типографика

| Роль | Шрифт | Начертание | Размер (mobile) |
|---|---|---|---|
| Заголовок экрана (вопрос/фраза) | Serif (с засечками, тёплый, не строгий) | Regular | 17–19px |
| Подзаголовок / пояснение | Sans-serif | Regular | 11–12px |
| Текст на кнопке | Sans-serif | Medium (500) | 13px |
| Подпись бренда | Sans-serif | Regular | 9–10px |

**Логика выбора:**
- Засечки (serif) на главной фразе дают ощущение личного письма, открытки — а не казённого интерфейса.
- Всё остальное (кнопки, подписи, технические элементы) — нейтральный sans-serif, чтобы не спорить с заголовком.
- Только два начертания во всём интерфейсе: обычное и среднее (500). Жирный (700) не использовать — он выглядит слишком тяжело для тёплой, мягкой темы.
- Текст всегда строчными (sentence case), без КАПСА — капс конфликтует с дружелюбным тоном.

---

## 3. Стиль иконок

- **Источник:** outline-иконки (тонкая линия, без заливки) — мягче и легче, чем закрашенные, подходят тёплой палитре.
- **Толщина линии:** тонкая, единообразная по всем иконкам.
- **Размер:**
  - Декоративные (сердечки на фоне) — 10–18px.
  - Внутри круглых плашек (аватар, статус) — 34–40px.
  - Внутри карточек мест — 16px.
- **Примеры использования:**
  - Сердечко — декоративный элемент на фоне, не несёт функции, используется в небольшом количестве (2-3 шт. на экран, не засорять).
  - Иконка кота/животного — аватар-заглушка, если фото не загружено.
  - Грустный смайл — состояние «отказа» на Экране 2.
  - Галочка в круге — финальное подтверждение/согласие.
  - Иконки мест (кофе, еда, дерево/парк и т.д.) — внутри карточек выбора места, по одной на карточку.
- **Цвет иконок:** всегда из палитры (коралл, персик, приглушённый коричневый) — никогда чёрный или серый по умолчанию.
- **Не использовать:** закрашенные (filled) иконки, многоцветные эмодзи, иконки с градиентом или тенью.

---

## 4. Компоненты

### Кнопки
- **Основная (Да / Готово / главное действие):** заливка коралл `#E8625A`, текст `#FBEAE6`, скругление 12px, без обводки.
- **Второстепенная (Нет / отмена):** прозрачный фон, текст приглушённо-коричневый `#B8857C`, тонкая обводка `#E8C9BC`, скругление 10px.
- Кнопки никогда не используют тень или градиент — только плоская заливка.

### Карточки выбора (например, места для свидания)
- Фон: белый.
- Обводка по умолчанию: тонкая, нейтральная (`var(--color-border-tertiary)`).
- Обводка при выборе: коралл, 1.5–2px.
- Внутри: маленькая квадратная плашка с иконкой (скругление 8px, фон из светлой палитры) + короткий текст названия.
- Скругление самой карточки: 12px.

### Аватар / заглушка фото
- Круг, фон — светлый персик `#FFD9C2`.
- Внутри — outline-иконка (например, кот) цветом коралл.
- Размер: 72–84px в зависимости от экрана.

### Декоративные элементы
- Парящие сердечки (outline, разных размеров и оттенков из палитры) — используются точечно, 2-3 на экран, не более. Они оживляют фон, но не должны спорить с текстом.

### Подпись бренда
- Всегда внизу экрана, мелким текстом, бледный беж `#C9B3A8`.
- На бесплатной/базовой версии — видна всегда. На премиум-версии — скрывается.

---

## 5. Скругления и отступы

- Карточка приглашения (контейнер «телефона»): радиус 20–22px.
- Кнопки: радиус 10–12px.
- Карточки выбора и плашки-иконки: радиус 8–12px.
- Внутренние отступы экрана: 16–20px по бокам.
- Отступ между элементами внутри карточки выбора: 8px.

---

## 6. Анимации (направление, без точных тайм-кодов)

- **Переход между экранами:** мягкое появление (fade + лёгкий сдвиг снизу вверх), без резких скачков.
- **Кнопка «Нет» (в простом сценарии №1):** убегает в случайную точку экрана и уменьшается при попытке нажатия; кнопка «Да» немного растёт с каждой попыткой.
- **Парящие сердечки:** медленное, едва заметное покачивание/всплытие на фоне — амбиентная анимация, не привлекающая основное внимание.
- **Финальный экран (согласие):** конфетти/сердечки коротким всплеском при появлении экрана, не зацикленно.
- **Общее правило:** анимация поддерживает эмоцию, но не отвлекает от текста и кнопок. Если сомневаешься — анимации должно быть меньше, не больше.

---

## 7. Что важно сохранить при масштабировании на другие темы

Эта тема («Тёплый персик») — основная и эталонная. При создании других палитр (премиум-бордовая, нейтральная для дня рождения и т.д.) сохраняется:
- та же структура ролей цвета (фон / акцент-декор / акцент-действие / текст основной / текст вторичный);
- тот же шрифтовой подход (serif для эмоционального заголовка + sans для всего остального);
- тот же стиль иконок (outline, без заливки);
- те же радиусы скругления и отступы;
- меняются только сами значения HEX и, при необходимости, иконка-заглушка аватара под тематику события (кот → кольца для свадьбы, торт для дня рождения и т.д.).
